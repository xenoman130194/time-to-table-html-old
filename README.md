# Time-SAP 0.6.0 RC

Краткое описание
-----------------
Time-SAP — веб-страница/инструмент для работы с расчётами времени и сопутствующими задачами (версия 0.6.0 Release Candidate). Файл интерфейса приложения: `Time-SAP 0.6.0 RC.html`.

Функции программы
-----------------
- Основные: ввод и обработка временных значений (часы, минуты), выполнение арифметики со временем (сложение, вычитание).
- Интерфейс: интерактивные элементы для ввода данных, кнопки управления расчётом и сбросом.
- Импорт/экспорт: (если реализовано) сохранение результатов локально через скачивание/копирование.
- Валидация: базовая проверка формата вводимых значений и предупреждения об ошибках ввода.

Требования и запуск
--------------------
- Открыть файл `Time-SAP 0.6.0 RC.html` в современном браузере (Chrome, Edge, Firefox).
- Нет серверных зависимостей — приложение работает полностью в браузере (если в проекте нет дополнительных скриптов/серверной части).

Технический долг
----------------
- Отсутствие модульных и интеграционных тестов — покрытие критичных вычислительных функций отсутствует.
- Ограниченная валидация ввода — сейчас проверка базовая; нужны edge-case тесты (нечисловые значения, очень большие числа, отрицательные значения).
- Локализация/интернационализация — тексты на русском, нет механизма переключения языков.
- Документация к коду — комментарии и описание алгоритмов в исходном HTML/JS минимальны.
- Отсутствие сборочного процесса и линтеров — нет автоматической проверки качества кода и форматирования.

Безопасность и конфиденциальность
---------------------------------
- XSS: если приложение принимает или отображает пользовательский ввод как HTML, необходимо экранировать/санитизировать вывод. Избегать вставки необработанного HTML.
- CSP: при развёртывании рекомендуются заголовки Content-Security-Policy для ограничения выполнения непреднамеренных скриптов.
- Отсутствие серверной части снижает риски, но если добавляются загрузки/сохранение файлов — валидировать пути и расширения.
- Не использовать `eval()` или динамическую интерпретацию кода строк, чтобы предотвратить выполнение вредоносного кода.
- Если реализуется обмен данными с сервером — хранить и передавать чувствительные данные по HTTPS и соблюдать минимизацию собираемых данных.

Лицензия
--------
Нет, для личного пользования
